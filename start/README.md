## 👀 파일확인

1. 해당 디렉토리로 이동

```
cd 파일 경로
```

2. 파일 리스트 출력  
  ✔️ `-al` 옵션은 숨겨진 파일 리스트도 보여준다.

```
ls -al
```



## 🌟 깃 초기화

1. 디렉토리 생성

```
mkdir 폴더명
```

2. 📌 깃 초기화

```
git init
```

3. 생성된 깃 파일 확인  
  ✔️  master branch 자동 생성됨
```
ls -al
```

## 🌟 깃 삭제

1. 디렉토리로 이동

```
cd 폴더 경로
```

2. 📌 깃 삭제

```
rm -rf .git
```

3. 파일 리스트 확인

```
ls -al
```
