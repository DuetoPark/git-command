## 🥕 log

```
> git log
```

- 커밋 내역을 보여준다.
- 가장 상단에 있을수록 최신 커밋
- `-p`: `diff`와 비슷하게 어떤 변화가 있었는지 내역을 보여준다.
- `--oneline`: log를 한줄로 표현한다.
