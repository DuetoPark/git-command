## 💡 깃에 tracking 하고싶지 않은 파일 처리

1. 프로젝트 root에 `.gitignore` 파일 생성
2. 파일명 넣기
  ```
  - 해당 파일만 처리하고 싶은 경우
    ᄂ 파일명
    ᄂ ex) log.log
  - 원하는 확장자만 처리하고 싶은 경우
    ㄴ *.확장자
    ᄂ ex) *.txt *.log 등등
  - 원하는 폴더의 모든 내용을 처리하고 싶은 경우
    ᄂ 폴더명/
    ᄂ ex) build/
  - 원하는 폴더 속에 일부 확장자만 처리하고 싶은 경우
    ᄂ 폴더명/*.확장자
    ᄂ ex) build/*.log
  ```
